# 메시지 매니저 (Message Manager)

SillyTavern용 메시지 관리 확장프로그램입니다.

## 기능 (개발 예정 목록)

- [ ] mes_button에 빠른 삭제 아이콘 추가
- [ ] 체크박스로 메시지 선택 → 삭제/숨기기
- [ ] 스와이프 메시지 관리 (목록 보기, 선택, 삭제)
- [ ] 채팅 번호로 빠른 스크롤 이동 (처음/이전/번호입력/다음/끝)
- [ ] 드래그 텍스트 빠른 수정
- [ ] 단어 찾기 및 바꾸기

## 설치 방법 (로컬 개발용)

1. SillyTavern이 설치된 폴더를 찾습니다.
2. 아래 경로에 이 폴더(`silly-message-manager`)를 통째로 복사해 넣습니다.
   ```
   SillyTavern/data/<내-사용자-핸들>/extensions/silly-message-manager
   ```
   (또는 "모든 사용자용으로 설치"를 원하면 `public/scripts/extensions/third-party/` 아래에 넣습니다.)
3. SillyTavern을 새로고침합니다.
4. 상단의 확장(퍼즐 조각/블록 아이콘) 메뉴에서 "메시지 매니저"가 목록에 보이고, 활성화(Enabled) 체크박스가 켜져 있는지 확인합니다.
5. 화면 우측 하단에 원형 버튼이 뜨면 설치 성공입니다.

## 개발 상태

현재는 뼈대(스켈레톤) 단계입니다. 버튼을 눌러도 아직 아무 기능이 없고,
브라우저 개발자 콘솔(F12)에 로그만 찍힙니다.
